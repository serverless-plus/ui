Serverless Plus UI Tool

## Usage

```bash
$ npm i @slsplus/ui --save
```

```js
import { startServer } from '@slsplus/ui';

startServer();
```

## Development

Install dependencies:

```bash
$ npm run bootstrap
```

Run server and client:

```bash
$ npm run dev
```
